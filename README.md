# 🔐 System Security Scanner

A standalone Python-based cybersecurity tool designed to perform basic system security assessments on a local machine.

## 🚀 Features

- 🔍 Localhost Port Scanning (20–1024)
- ⚙️ Running Process Enumeration
- 🔑 Password Strength Analysis

## 🧠 Purpose

This project simulates a basic SOC (Security Operations Center) workflow by identifying potential security risks such as exposed ports and weak authentication practices.

## 🛠️ Technologies Used

- Python
- Socket Programming
- OS Command Integration

## ▶️ How to Run

```bash
python scanner.py
```
